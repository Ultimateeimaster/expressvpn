# ExpressVPN Privacy Setup Checklist

A quiet setup page for keeping VPN preferences and privacy checks in one place.

Use it to remember preferred regions, DNS notes, leak-check routines, and travel profiles. It is a checklist, not a replacement for the actual VPN service.

The main page is here:

## What you can use it for

- Region profile notes
- DNS and leak-check routine
- Travel setup checklist
- Privacy settings reminder

## Why this repo is small

Not every project needs a heavy codebase. This repo is here to give the topic a
clean public home on GitHub, keep the Mirrorify domain easy to find, and leave a
simple place for repository metadata.

Unofficial page only. No accounts, credentials, cracked clients, or bypass instructions.

That's the whole point: clear page, clean files, no clutter.
